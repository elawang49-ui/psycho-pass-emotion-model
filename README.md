# 心理测量者 PSYCHO-PASS

心理测量者（PSYCHO-PASS）是一个基于罗素情绪圆环模型的中文短文本情绪检测与舆情情绪监测实验项目。

系统可以根据中文文本生成对应的 `valence`（愉悦度）、`arousal`（唤醒度）、情绪方向、情绪强度与时间戳记录，用于观察评论区、用户反馈、观众弹幕等文本场景中的情绪变化趋势。

本项目当前版本聚焦于中文情绪识别与情绪罗盘可视化，后续可能扩展到关系风险分析、情绪时间线、反讽识别与运营监测等方向。

## 项目定位

本项目当前版本聚焦于中文短文本情绪量化与可视化，适用于评论区文本、用户反馈、观众弹幕、社交文本等场景的情绪趋势观察。

本项目不是心理诊断工具、医疗工具或关系决策工具。  
它更接近一个轻量级的中文情绪识别 / 舆情情绪监测 / 文本分析实验原型。

后续版本可能扩展到关系风险分析、情绪时间线、低成本亲密风险、责任转移、反讽识别等方向。

本项目采用轻量级本地机器学习方案进行最终情绪预测。系统运行与推理阶段不调用在线大模型 API，也不依赖 transformers 或 HuggingFace。

## 罗素圆环模型

罗素圆环模型使用二维空间描述情绪：

- `valence`：愉悦度，范围为 `-1` 到 `1`。越接近 `1` 越正向，越接近 `-1` 越负向。
- `arousal`：唤醒度，范围为 `-1` 到 `1`。越接近 `1` 越激动、高唤醒，越接近 `-1` 越平静、低唤醒。

系统将二维坐标进一步解释为八方向情绪：

- 开心 / 满足
- 兴奋 / 狂喜
- 紧张 / 警觉
- 愤怒 / 焦灼
- 失望 / 厌烦
- 悲伤 / 疲惫
- 麻木 / 平静
- 放松 / 安心

对于接近圆心的低强度文本，系统会显示为低波动倾向，避免把轻微文本强行贴上强情绪标签。

## 功能列表

- 单句中文情绪检测
- 批量 CSV / Excel 情绪检测
- 时间轨迹分析
- 八方向情绪罗盘可视化
- `pred_valence` / `pred_arousal` 输出
- 情绪角度、强度、强度等级、主情绪标签输出
- 极短词兜底规则
- 反讽 / 阴阳怪气兜底规则
- 预测结果 CSV 下载
- 测试集评估与 badcase 分析脚本
- 数据清洗与无效标签隔离脚本

## 技术路线

本项目采用轻量级本地机器学习方案进行最终情绪预测。系统运行与推理阶段不调用在线大模型 API，也不依赖 transformers 或 HuggingFace。

数据构建阶段使用在线大模型辅助生成部分匿名化中文短文本样本，随后经过人工规则约束、格式清洗、隐私脱敏、测试集评估与 badcase 迭代。在线大模型仅用于语料生成辅助，不参与最终模型推理。

技术栈包括：

- TF-IDF 字符级 ngram
- RandomForestRegressor
- MultiOutputRegressor
- Streamlit
- pandas
- matplotlib
- scikit-learn

## 数据说明

完整实验数据约 6k 条，包含自建中文短文本样本和部分在线大模型辅助生成样本。出于隐私和数据伦理考虑，仓库暂不公开完整训练数据，仅提供 `demo_data.csv` 展示数据格式和本地运行流程。

训练数据字段：

```text
text,valence,arousal,emotion
```

测试数据字段：

```text
text,expected_valence_min,expected_valence_max,expected_arousal_min,expected_arousal_max,expected_emotion,case_type
```

- 中性日常陈述：86.67%

## 当前局限

- 数据规模仍较小，测试结果对样本分布较敏感。
- 对复杂语境、转折语气、隐含情绪的理解能力有限。
- 一些类别存在相互牵引，例如补强高唤醒样本可能影响中性或低唤醒样本。
- 当前模型是传统机器学习回归模型，不具备大型语言模型的语义推理能力。
- demo 数据仅用于展示格式，不代表完整训练分布。

## 在线大模型使用说明

本项目最终推理过程不调用在线大模型；在线大模型仅用于数据构建阶段的语料辅助生成。

具体来说，部分训练语料曾借助在线大模型生成匿名化中文短文本样本，之后经过格式清洗、规则约束、隐私筛查、测试集评估和 badcase 迭代。最终运行时使用的是本地 `TF-IDF + MultiOutputRegressor + RandomForestRegressor` 模型。
